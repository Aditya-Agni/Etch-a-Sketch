# 🎨 Etch-a-Sketch

A browser-based Etch-a-Sketch built using **vanilla JavaScript**, with a strong focus on **DOM manipulation** and interactive UI behavior.

---

## 🚀 Live Demo
👉 https://aditya-agni.github.io/Etch-a-Sketch/

---

## 📌 About

This project replicates the classic Etch-a-Sketch experience in the browser.  
It focuses on building dynamic interfaces using pure JavaScript without any libraries.

---

## 🧠 Key Concepts

- **DOM Manipulation** – dynamically creating and updating grid elements  
- **Event Handling** – hover-based drawing and user interactions  
- **JavaScript Logic** – grid generation, reset functionality, and UI updates  

---

## ⚙️ Features

- 🎯 Dynamic grid  
- 🎨 Hover drawing effect  
- 🔄 Clear/reset option  
- 📏 Adjustable grid size  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 💡 Learning Outcome

Improved understanding of DOM manipulation, event-driven programming, and building interactive web applications using vanilla JavaScript.
